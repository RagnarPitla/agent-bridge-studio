# Roadmap

## MVP

- Detect `gh`, `copilot`, `gh copilot`, and `claude`.
- Adapter interface.
- Copilot CLI adapter.
- Claude Code adapter.
- Worktree-scoped execution.
- Manual approval before PR creation.

## V1

- UI selector for agent per task.
- Project default agent.
- Agent capability matrix.
- Host-aware GitHub Enterprise support.
- Structured task timeline.

## V2

- Agent benchmarking per task type.
- Multi-agent split/merge workflows.
- Copilot for implementation + Claude for QA review option.
- Claude for planning + Copilot for code execution option.
- Policy profiles for trusted vs cautious modes.
